Path to Configuration

The path to the CodecheckConfiguration.ini.

What's this file?

This file contain the list of the checkers used for a codeCheck analysis.

This file will be created by using Understand interface.

After selecting the list of the checkers, this configuration will need to be exported

In this field you need to indicate the path of this file

Note: The path of this file is required if you want to create a codeCheck report.
